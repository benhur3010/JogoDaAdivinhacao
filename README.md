# JogoDaAdivinhacao

Projeto Stage 05 do Explorer Rocketseat.

Início dos estudos de JavaScript, desenvolvendo um projeto que gera um número aleatório e requisita ao usuário acertá-lo. Quando a condição se torna verdadeira é disparado um alerta que informa o acerto e a quantidade de tentativas que foram necessárias.
## Stack utilizada

**Front-end:** HTML / CSS / JavaScrip
